# 📊 Supply Chain Performance & Inventory Optimization Dashboard (Power BI)

This project is my second Power BI dashboard focused on analyzing supply chain operations and inventory performance. It aims to transform raw operational data into meaningful insights that support efficient decision-making.

---

## 🎯 Objective
To analyze supply chain data and identify patterns in inventory, transportation, and order fulfillment in order to improve operational efficiency.

---

## 📁 Dataset
The dataset includes:

- Order details
- Product and category information
- Inventory levels
- Supplier and logistics data
- Transportation cost and lead time
- Sales and order status

---

## 📊 Key KPIs
- Warehouse Utilization (%)
- Inventory Turnover Ratio
- Days of Inventory
- Total Sales & Orders

---

## 📈 Analysis Performed
- Inventory level tracking and stock analysis  
- Supplier and transportation performance evaluation  
- Demand trends and sales analysis over time  
- Backorder and order status monitoring  
- Region-wise performance comparison  

---

## 🔍 Key Insights
- Certain regions show higher transportation costs, indicating optimization opportunities  
- Inventory turnover suggests moderate stock movement with scope for improvement  
- Backorders highlight demand-supply gaps in specific product categories  
- Sales trends vary across years, indicating demand patterns  
- Warehouse utilization suggests potential for better capacity planning  

---

## 🛠 Tools Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (Data Cleaning & Transformation)  

---

## ⚙️ Features
- Interactive dashboard with slicers (Region, Category)  
- KPI cards for performance tracking  
- Time-based trend analysis  
- Multi-dimensional analysis (Region, Product, Supplier)  

---

## 🧠 Skills Demonstrated
- Data Cleaning  
- Data Modeling  
- Data Visualization  
- Business Analysis  
- KPI Development using DAX  

---

## 📌 Conclusion
This project demonstrates how supply chain data can be analyzed to identify inefficiencies and improve business decisions through data-driven insights.
